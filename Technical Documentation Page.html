    What is HTML? According to the dictionary HTML is "Hypertext Markup Language, a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages." It's not really a lot of information for the defintion, but we can learn that it does stuff for the world wide web. Essentially, HTML is a system that describes a structure and elements within a web page using markup. Using what are called tags you would establish an element and the information or text that goes into that element. For example common tags are "heading", "paragraph", "table", etc.These tags are used to structure your webpage and define elements within the page to organize everything. If you're itching to see some actual HTML here's an example of HTML using the header tag. Open up your notepad and start with this. Open and close the first tag with the less then and greater than sign then type in the word header in between the two signs. Now type in hello world. Next type in an another greater than and lesser than sign. Now add a forward slash and header inside of those signs. Your code should look like this now:

    Hello World

Underneath the header line, using the same signs create a paragraph element by placing a p in between the two sets of greater than and lesser than signs In between the two sets type in any paragraph that you want.     CSS

    Ok great! You've got the simple basics for HTML. You know that it's used for structuring a webpage and elements within it. Great. So how do you style it and make everything pretty and professional? Well that is where CSS comes in. CSS, or cascadding style sheets, is a language that describes the style of an HTML document. CSS tells us how the webpage will look. So think of HTML as the home builder. Think of CSS as the designer. You create the CSS in a style section. The style section is defined by the tags of style. In between the two tags you'd have a number of elements that you're seeking to design and make pretty. Now lets get to the coding! How would you go about doing this? Well you do this by creating a style section for your coding. It also depends on how you broke up your HTML. You design and use CSS based off the strucutre you've created. So for example if you created a paragraph and wanted to style it in a verdana font at 12 font size. Well then you would put in your style section of you coding: p { font-family: verdana; font-size: 20px; }

    Great! Now you've styled your paragraphs in that font-size and font-family. What else can you do? Well one other aspect of CSS is you can give color to the page. Black and white is great but let's add a background to our imaginary page. We do this by styling the body element of the webpage so for example it would look like this in our style section: body { background-color: green; }
